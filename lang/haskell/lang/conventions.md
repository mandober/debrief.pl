# Conventions and rules

<!-- #region Conventions -->

Rules
- PascalCase: ModuleName
- camelCase: varName, functionName

Conventions
- `/app` dir 
  - `/app/Main.hs` file/module
- `/src` dir for source files
  - `/src/Lib.hs` file
- `/test` dir for tests
  - `/test/Spec.hs` file

<!-- #endregion -->
